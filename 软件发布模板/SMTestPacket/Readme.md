## 产测软件说明



### 1.打开方式

​		三种打开方式：

打开SMShell.exe，遇到错误会停止运行

打开DEBUG.bat，遇到错误继续运行，测试用这个即可

打开AutoRun.bat，自动测试才会使用，正常不用

### 2.测试配置

   点击login，输入fae进入测试配置界面。

   在Data Test Items界面勾选要测试的项目，勾选后到Basic Set界面对各个参数的卡控范围进行配置，配置完成后，切换界面即可保存配置

### 3.测试结果

  测试结果会在测试结束后打印在右侧，也会同时保存在Test Result文件夹里，目前没有烧录ID，会按时间命名保存

### 4.开发板更新

产测软件和调试软件所使用固件不一样，用调试软件烧录本文件夹所附的更新BIN文件，成功后会看到红色LED在闪烁。
